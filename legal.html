Politique de confidentialité

Dernière mise à jour : juin 2026

1. Qui sommes-nous ?

OrionBot est un service de bot Discord développé et maintenu par XeFoXx. Le site web associé est accessible à l'adresse orionbot-backend-hxyh.onrender.com.

2. Données collectées

Lors de la connexion via Discord OAuth2, nous collectons les données suivantes :

  • Votre identifiant Discord (ID)
  • Votre pseudo Discord
  • Votre avatar Discord
  • Votre adresse email associée à votre compte Discord
  • La liste des serveurs Discord dont vous êtes administrateur

3. Utilisation des données

Ces données sont utilisées uniquement pour :

  • Vous permettre de vous connecter au dashboard OrionBot
  • Afficher vos serveurs Discord dans le dashboard
  • Vous permettre de configurer OrionBot sur vos serveurs

4. Stockage des données

Vos données de session sont stockées de manière sécurisée dans une base de données MongoDB hébergée sur MongoDB Atlas. Les sessions expirent automatiquement après 7 jours.

5. Partage des données

Vos données ne sont jamais vendues, partagées ou transmises à des tiers. Elles restent strictement confidentielles et utilisées uniquement dans le cadre du service OrionBot.

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès à vos données
  • Droit de rectification
  • Droit à l'effacement (droit à l'oubli)
  • Droit d'opposition

Pour exercer ces droits, contactez-nous sur notre serveur Discord : discord.gg/zgs4ubTNdZ

7. Cookies

OrionBot utilise uniquement un cookie de session technique nécessaire au fonctionnement du service. Aucun cookie publicitaire ou de traçage n'est utilisé.

8. Contact

Pour toute question relative à vos données personnelles, rejoignez notre serveur Discord : discord.gg/zgs4ubTNdZ
